# Java-Banking-System
Version 2 of a Java Banking System built in Eclipse using Java 

## Features
- Account Creation
- Deposits
- Withdrawals
- Balance Tracking
- Savings Account
- Account Transfers
- Overdrawn Balance Warning

## Technologies Used
- Java
- Eclipse IDE
- Object-Oriented Programming (OOP)

## Future Improvements 
- Transaction History
- User Authentication System
- File Saving System
- Interest Calculation
- Database Integration
